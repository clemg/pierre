### Package Exports [toc-ignore]

The package provides several entry points for different use cases:

| Package                        | Description                                                                                                |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------- |
| `@pierre/trees/path-store`     | [Path-store API](#path-store-api) — imperative file tree with mutations, always-on virtualization, and SSR |
| `@pierre/trees`                | [Vanilla JS API](#vanilla-js-api) and [utility functions](#utilities) for building file trees (legacy)     |
| `@pierre/trees/react`          | [React component](#react-api) for rendering file trees with full interactivity (legacy)                    |
| `@pierre/trees/ssr`            | [Server-side rendering utilities](#ssr) for pre-rendering the tree with fast first paint (legacy)          |
| `@pierre/trees/web-components` | Registers the `<file-tree-container>` custom element and shadow DOM helpers                                |
