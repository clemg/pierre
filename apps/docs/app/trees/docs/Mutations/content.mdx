## Mutations

`PathStoreFileTree` supports native tree mutations — you can add, remove, move,
and batch operations directly on the tree without replacing the entire file
list. All mutations update the internal store and re-render the tree
automatically.

### Mutation Events

Listen for mutation events with `onMutation`. The handler receives a typed event
object describing what changed. Pass `'*'` to listen for all mutation types, or
a specific type (`'add'`, `'remove'`, `'move'`, `'batch'`, `'reset'`).

`onMutation` returns an unsubscribe function.
