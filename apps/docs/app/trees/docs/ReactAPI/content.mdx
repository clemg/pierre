## React API

<Notice icon={<IconInfoFill />}>
  The path-store API is imperative — use it in React via `useEffect` and
  `useRef`. The legacy `FileTree` React component is still available from
  `@pierre/trees/react`.
</Notice>

`PathStoreFileTree` does not expose a React component. Instead, you create an
instance imperatively and mount it into a DOM node. This pattern works well with
SSR: pre-render on the server, send the HTML, and hydrate on the client.
