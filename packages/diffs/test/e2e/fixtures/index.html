@pierre/diffs — E2E fixtures

These are the standalone pages the Playwright suite drives against the built /dist. Open one and interact with it exactly as a test would. Each fixture sets a readiness flag on window once its content has rendered.

  • diff-render.html

    A FileDiff (split view) with two changed regions and a collapsed unchanged block. Buttons toggle split/unified and reject the first hunk. Drives diff-render.pw.ts (rendering, split/unified, expand context, accept/reject). Ready flag: window.__diffReady.

  • edit.html

    A FileDiff attached to the Editor (edit mode). The additions column is editable; deletions are read-only. Drives edit.pw.ts (typing, onChange, undo/redo). Ready flag: window.__editReady; edits are logged to window.__editorEvents.

  • edit-collapsed.html

    A FileDiff + Editor over a 60-line file whose unchanged gap collapses. Drives edit-collapsed.pw.ts (fold-skip navigation, reveal on search, separator expansion mid-edit, session exit). ?diffStyle=unified switches views. Ready flag: window.__editReady.

  • conflict.html

    An UnresolvedFile with two merge conflicts. Clicking the accept-current/incoming/both buttons resolves each conflict. Drives conflict.pw.ts. Ready flag: window.__conflictReady.

  • file-states.html

    An added file (oldFile: null) and a deleted file (newFile: null), showing additions-only and deletions-only rendering. Drives file-states.pw.ts. Ready flag: window.__fileStatesReady.

  • editable.html

    A single editable File + Editor. Drives find-replace.pw.ts (Cmd/Ctrl+F, Replace All), multi-edit.pw.ts (multi-caret, Tab/Shift+Tab), and edit-rewrite.pw.ts (select-all rewrite). Ready flag: window.__editableReady.

  • markers.html

    An editor with setMarkers squiggles (error/warning/info), hover popups, and boundary containment. Drives markers.pw.ts. Ready flag: window.__markersReady.

  • line-select.html

    A read-only File with line selection and the gutter utility button. Drives line-select.pw.ts (click/drag selection, utility click). Ready flag: window.__lineSelectReady.

  • annotations.html

    A File with a custom inline renderAnnotation row anchored to a line. Drives annotations.pw.ts. Ready flag: window.__annotationsReady.

  • theme.html

    An editor that toggles between the dark and light themes; verifies token colors change and a selection survives the switch. Drives theme.pw.ts. Ready flag: window.__themeReady.

  • selection-action.html

    An editor with enabledSelectionAction and a custom renderSelectionAction popover; verifies it appears on a selection, stays in bounds, and reports the selected text. Drives selection-action.pw.ts. Ready flag: window.__selectionActionReady.

  • selection-action-edges.html

    A taller 16-line editor with enabledSelectionAction used to drag forward/backward selections whose head sits mid-file or on the first/last row; verifies the popover flips off the code element's own clip box and never spills past the file component's edges. Drives selection-action.pw.ts. Ready flag: window.__selectionActionEdgesReady.
